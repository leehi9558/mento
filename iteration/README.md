# Ex1

배열 `team1`의 모든 이름을 순서대로 콘솔에 출력하는 for 반복문을 작성하세요.

```

const team1 = ['박예성', '이해인', '임지환'];

// 예상 출력:
// 박예성
// 이해인
// 임지환

```

# Ex2

배열 `team1`을 순회하면서, 이름에 '박'이 들어가지 않은 이름만 콘솔에 출력하세요.

```

const team1 = '['박예성', '이해인', '임지환'];

// 예상 출력:
// 이해인
// 임지환

```

# Ex3

배열 `team1`의 각 이름 뒤에 ' 멘티'를 붙여 하나의 긴 문자열로 만들고 출력하세요.

```

const team1 = ['박예성', '이해인', '임지환'];

// 예상 출력:
// 박예성 멘티, 이해인 멘티, 임지환 멘티

```

# Ex4

객체 `team1`을 `for~in` 반복문으로 순회하며, key(팀원 이름)를 출력하세요. 만약 출력 값이 '이해인'일 경우, 이름 대신 '스프린트'를 출력하세요.

```

const team1 = {
  '박예성': 1,
  '이해인': 2,
  '임지환': 3
};

// 예상 출력:
// 박예성
// 스프린트
// 임지환

```
